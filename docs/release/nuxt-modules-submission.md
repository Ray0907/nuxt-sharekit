# Nuxt Modules submission

The candidate database entry is `docs/release/nuxt-modules.yml`. The npm package
and public docs URL are live. To request the listing:

1. Open the `Module Listing Request` issue form in
   [`nuxt/modules`](https://github.com/nuxt/modules/issues/new/choose).
2. Include the description, repository, npm package, and Nuxt 4 compatibility.
3. Link the public documentation and the clean-install verification evidence.
4. After acceptance, compare the generated database entry with
   `docs/release/nuxt-modules.yml`.

The Nuxt database automatically syncs npm metadata, so the package must remain
public while the request is reviewed.
